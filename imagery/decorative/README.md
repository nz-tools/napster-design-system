# Decorative

**Slot:** Atmospheric — Brand Feeling

## What's in this folder

Non-photographic background elements, geometric boxes, gradient panels, and the grain texture. These are the connective tissue between sections and the canvas behind hero copy.

## Treatment

`napster-grain-texture.jpg` is the system-wide tactile overlay (mix-blend-mode: screen, ~18% opacity). `napster-element-pink-gradient.avif` and the `*-box.avif` set carry the Beam aesthetic.

## Contents

- Hero backgrounds and CTAs
- Trust band visuals
- Geometric layout elements (left, right, logo-flanking boxes)
- The grain texture for tactile depth
